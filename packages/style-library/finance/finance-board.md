# Finance Board Report — Style Specification
**Version:** 1
**Industry:** Finance

## Typography
- Heading font: Garamond Premier Pro
- Body font: Garamond
- Heading sizes: H1 28pt bold, H2 20pt bold, H3 16pt semibold, H4 13pt semibold
- Body size: 11.5pt
- Line height: 1.45
- Financial figure size in callouts: 36pt Garamond bold
- Table text: 10pt
- Footnote and disclaimer size: 8.5pt

## Color palette
- Primary: #0D1B2A — use for headings, primary chart series, KPI callout numbers, and section dividers
- Secondary: #1B3A5C — use for subheadings, secondary chart series, and table headers
- Accent: #C9A84C — use for positive performance indicators, quarter highlights, and executive summary borders
- Alert: #9B2226 — use for negative variances, risk flags, and below-target indicators
- Body text: #1C1C1C
- Background: #FFFFFF
- Executive summary background: #F8F6F0
- Table header background: #0D1B2A with white text
- Table alternating rows: #FAFAF8
- Chart palette: #0D1B2A, #1B3A5C, #4A7C9B, #C9A84C, #7B9E6B

## Layout rules
- Page size: 8.5in x 11in (US Letter)
- Margins: 0.9in top, 0.9in bottom, 1in left, 1in right
- Section spacing: 24pt before H1, 16pt before H2, 10pt before H3
- Paragraph spacing: 7pt after each paragraph
- Column structure: single column for narrative; two-column for KPI dashboard pages
- Page numbers: bottom right, 9pt, "Page X" format
- Header: "BOARD REPORT — CONFIDENTIAL" left, reporting period right, 9pt, #0D1B2A rule below
- Footer: company name left, "Prepared for Board of Directors" center, date right, 8pt #999999
- Cover page required: company name, "Board of Directors Report," reporting period, date, preparer

## Formatting conventions
- KPI dashboard: first content page after cover — grid of 4-6 key metrics in large-format callout boxes
- Each KPI box: metric name (9pt label), value (36pt bold), trend arrow, vs. prior period comparison
- Positive variances: #7B9E6B with upward arrow; Negative variances: #9B2226 with downward arrow
- Financial tables: right-aligned numbers, thousands separator, negative values in parentheses and red
- Basis point and percentage changes: always include direction indicator (+/-) and color coding
- Charts: clean axis labels, data labels on key points, legend below chart, grid lines #E8E8E8
- Waterfall charts for variance analysis using primary and alert colors
- Executive summary: #F8F6F0 background block with gold left border (#C9A84C, 4pt)
- Appendices for detailed schedules, numbered (Appendix A, B, C)
- Disclaimers and forward-looking statement warnings in footer-style text, 8.5pt italic

## Tone and voice
- Authoritative, measured, and fiduciary in nature
- Write as a CFO presenting to independent board members and audit committee
- Lead with performance summary, then drill into detail
- Factual and data-driven — every statement supported by specific figures
- "The Company reported," "Performance reflects," "Management expects"
- Neutral on negative results — state the fact and the corrective plan, do not apologize
- Forward-looking statements clearly labeled: "Management anticipates" or "Projections indicate"
- Risk discussion balanced: identify risk, quantify potential impact, describe mitigation
- No marketing language — this is fiduciary communication

## What this style MUST produce
- Board-ready financial documents with professional gravitas
- Cover page with company name, report title, and period
- KPI dashboard as the first content page
- Large-format metric callouts with trend indicators and period comparisons
- Financial tables with proper formatting (right-aligned, parenthetical negatives, red for losses)
- At least one variance analysis (waterfall chart or bridge) per financial section
- Executive summary with gold accent border
- Forward-looking statement disclaimer where projections appear
- Page count: per deliverable type specification
- All sections in the order specified by the deliverable module

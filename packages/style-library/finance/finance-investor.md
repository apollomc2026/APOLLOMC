# Finance Investor — Style Specification
**Version:** 1
**Industry:** Finance

## Typography
- Heading font: Helvetica Neue
- Body font: Helvetica Neue Light
- Heading sizes: H1 26pt semibold, H2 20pt medium, H3 15pt medium, H4 12pt semibold
- Body size: 11pt
- Line height: 1.5
- Metric callout size: 40pt Helvetica Neue bold
- Caption and source text: 9pt
- Disclaimer text: 8pt

## Color palette
- Primary: #2D6A4F — use for headings, primary chart series, growth indicators, and key metric callouts
- Secondary: #40916C — use for secondary chart fills, progress bars, and subheading accents
- Accent: #95D5B2 — use for positive trend backgrounds, success callouts, and supporting chart series
- Neutral: #6C757D — use for comparative baselines, benchmark lines, and secondary text
- Alert: #D62828 — use only for negative indicators and risk warnings
- Body text: #212529
- Background: #FFFFFF
- Highlight section background: #F0FAF4
- Table header background: #2D6A4F with white text
- Table alternating rows: #F8FBF9
- Chart palette: #2D6A4F, #40916C, #95D5B2, #74C69D, #6C757D

## Layout rules
- Page size: 8.5in x 11in (US Letter)
- Margins: 0.85in top, 0.85in bottom, 0.9in left, 0.9in right
- Section spacing: 22pt before H1, 16pt before H2, 10pt before H3
- Paragraph spacing: 8pt after each paragraph
- Column structure: single column for narrative; two-column for side-by-side metrics and comparisons
- Page numbers: bottom center, 9pt
- Header: company logo placeholder left, document type right, thin green rule (#2D6A4F, 1.5pt)
- Footer: legal disclaimer line centered, 8pt #AAAAAA
- No cover page required — lead with content immediately

## Formatting conventions
- Growth metrics front and center: large-format callouts (40pt number) with period-over-period delta
- Positive deltas: green (#2D6A4F) with upward arrow; Negative: red (#D62828) with downward arrow
- Revenue and user growth charts: area charts with gradient fill (primary to accent)
- Milestone timeline: horizontal format showing key achievements with dates and context
- Financial projections in structured table: quarterly columns, clearly labeled as projected vs. actual
- Use of forward-looking icons (rocket, chart-up) acceptable in milestone sections
- Key investor highlights in green-bordered callout boxes (#F0FAF4 background)
- Capitalization table or ownership summary in clean matrix format
- Unit economics presented as a funnel or waterfall visualization
- Sources and methodology footnotes for all projected figures

## Tone and voice
- Confident, growth-oriented, and transparent
- Write as a portfolio manager communicating to limited partners or institutional investors
- Lead with performance and momentum before addressing challenges
- Honest about headwinds: "Revenue growth decelerated to 18% due to" — state cause directly
- Use growth language naturally: "momentum," "trajectory," "runway," "inflection point"
- Quantify everything: ARR, MRR, burn rate, months of runway, customer LTV, CAC payback
- "The portfolio company delivered," "Performance reflects," "We project"
- Avoid hype — let the numbers convey the story
- Risk discussion always paired with mitigation strategy
- Forward-looking statements flagged: "Based on current trajectory, we project"

## What this style MUST produce
- Investor-grade communications that build confidence through data transparency
- Growth metrics as the first visual element (large-format callouts)
- Period-over-period comparisons with directional indicators
- At least one growth visualization (area chart or line chart) per major financial metric
- Key milestones in timeline format
- Clean financial tables with clear actual vs. projected labeling
- Green accent elements on every page reinforcing growth theme
- Forward-looking statement disclaimers where projections appear
- Page count: per deliverable type specification
- All sections in the order specified by the deliverable module

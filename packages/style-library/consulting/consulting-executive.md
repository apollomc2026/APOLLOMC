# Consulting Executive — Style Specification
**Version:** 1
**Industry:** Consulting

## Typography
- Heading font: Georgia
- Body font: Georgia
- Heading sizes: H1 30pt bold, H2 22pt bold, H3 16pt semibold, H4 13pt semibold italic
- Body size: 11.5pt
- Line height: 1.5
- Caption and label size: 9pt
- Chart annotation size: 8.5pt

## Color palette
- Primary: #0A2463 — use for headings, key metric callouts, chart primary series, and section dividers
- Secondary: #1E5288 — use for subheadings, secondary chart series, and table headers
- Accent: #D4AF37 — use for highlight indicators, key figure underlines, and executive summary borders
- Body text: #1C1C1C
- Background: #FFFFFF
- Callout background: #F4F6FA
- Table header background: #0A2463 with white text
- Table alternating rows: #F8F9FB
- Chart palette: #0A2463, #1E5288, #3A7CA5, #D4AF37, #81B29A

## Layout rules
- Page size: 8.5in x 11in (US Letter)
- Margins: 0.85in top, 0.85in bottom, 1in left, 1in right
- Section spacing: 24pt before H1, 18pt before H2, 12pt before H3
- Paragraph spacing: 8pt after each paragraph
- Column structure: single column for narrative; two-column permitted for key metrics dashboards
- Page numbers: bottom right, 9pt, "Page X" format
- Header: company logo placeholder left, document classification right, separated by #0A2463 rule (2pt)
- Footer: thin gold (#D4AF37) accent line, confidential notice centered, 8pt #888888
- Cover page: required for all documents over 5 pages — title centered vertically, client name, date, prepared by

## Formatting conventions
- Executive summary box on page 1 or 2: bordered with #D4AF37 left accent (4pt), #F4F6FA background
- Key metrics presented in large-format callout numbers: 36pt bold #0A2463 with label below in 9pt
- Section numbering: Roman numerals for top level (I, II, III), decimal for subsections (1.1, 1.2)
- Recommendations formatted as numbered action items in a distinct box
- All financial figures right-aligned in tables, formatted with commas and appropriate currency symbols
- Charts preferred over tables where data supports visualization
- Appendices numbered separately (Appendix A, B, C)
- Source citations as footnotes, 9pt, gray text

## Tone and voice
- Authoritative, measured, and commanding
- Write as a senior partner addressing a board of directors
- Lead every section with the conclusion or recommendation, then provide supporting evidence
- Use "we recommend," "our analysis indicates," "the evidence supports"
- Avoid hedging unless genuinely uncertain — state findings with confidence
- No jargon without immediate definition in parentheses
- Quantify everything possible: "revenue increased 23%" not "revenue increased significantly"
- Third person for the client organization; first person plural for the consulting engagement

## What this style MUST produce
- Boardroom-ready documents that convey authority and analytical depth
- Cover page for documents exceeding 5 pages
- Executive summary within the first 2 pages with gold accent border
- At least one key metrics callout per major section (large-format numbers)
- Numbered action items in a visually distinct recommendations section
- Professional chart styling using the defined color palette
- All financial data properly formatted and right-aligned
- Page count: per deliverable type specification
- All sections in the order specified by the deliverable module

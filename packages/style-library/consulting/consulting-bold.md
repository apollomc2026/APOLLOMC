# Consulting Bold — Style Specification
**Version:** 1
**Industry:** Consulting

## Typography
- Heading font: Montserrat
- Body font: Open Sans
- Heading sizes: H1 32pt extra-bold, H2 24pt bold, H3 18pt semibold, H4 14pt semibold
- Body size: 11pt
- Line height: 1.55
- Pull quote size: 20pt Montserrat light italic
- Stat callout size: 48pt Montserrat extra-bold
- Caption size: 9pt

## Color palette
- Primary: #E76F51 — use for headings, primary callouts, key action items, and chart emphasis
- Secondary: #264653 — use for body section headers, secondary chart series, and footer
- Accent: #2A9D8F — use for success indicators, positive metrics, and supporting callouts
- Tertiary: #E9C46A — use for warning states, attention badges, and highlight backgrounds
- Body text: #264653
- Background: #FFFFFF
- Hero section background: #264653 with white text
- Callout background: #FFF3EE
- Table header background: #E76F51 with white text
- Table alternating rows: #FEF9F6

## Layout rules
- Page size: 8.5in x 11in (US Letter)
- Margins: 0.75in top, 0.75in bottom, 0.85in left, 0.85in right
- Section spacing: 30pt before H1, 20pt before H2, 14pt before H3
- Paragraph spacing: 8pt after each paragraph
- Column structure: flexible — full-width for narrative, two-column for impact metrics, three-column for feature comparisons
- Page numbers: bottom right, 10pt bold, #E76F51
- Header: bold orange (#E76F51) bar (4pt) across top of every page
- Footer: dark (#264653) bar with white page number and document title
- Hero blocks: full-width #264653 background sections for major transitions

## Formatting conventions
- Impact statements in large-format stat blocks: 48pt number + descriptor below in 11pt
- Key takeaways in orange-bordered boxes with #FFF3EE background
- Icon usage encouraged: checkmarks, arrows, warning triangles for quick visual scanning
- Before/after comparisons in side-by-side panels with contrasting backgrounds
- Progress bars and percentage indicators for completion or growth metrics
- Section dividers: full-width #E76F51 rule (3pt) between major parts
- No Roman numerals — use bold decimal numbering (1, 2, 3) or no numbering at all
- Bullet points: filled orange circles for primary, gray circles for secondary
- Call-to-action boxes: #E76F51 background, white bold text, centered

## Tone and voice
- Energetic, confident, and action-oriented
- Write as a transformation advisor who sees opportunity everywhere
- Lead with impact: "This initiative will save $2.4M annually" not "We have identified potential savings"
- Short, punchy sentences for emphasis. Longer sentences for explanation.
- Use imperatives for recommendations: "Implement," "Launch," "Accelerate," "Prioritize"
- Rhetorical questions permitted to engage readers: "What would 30% faster delivery mean for your pipeline?"
- Superlatives acceptable when supported by data
- First person plural ("we") for engagement team; second person ("you," "your team") for client
- Bold claims backed by bold evidence

## What this style MUST produce
- High-energy documents that feel more like a pitch than a report
- At least two large-format stat callouts per document (48pt numbers)
- Hero sections with dark backgrounds for major document transitions
- Orange accent elements visible on every page
- Impact-first writing: conclusions and numbers before explanations
- Before/after or current-state/future-state comparisons where relevant
- Call-to-action elements for all recommendation sections
- Page count: per deliverable type specification
- All sections in the order specified by the deliverable module

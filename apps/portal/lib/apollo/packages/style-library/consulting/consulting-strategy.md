# Consulting Strategy — Style Specification
**Version:** 1
**Industry:** Consulting

## Typography
- Heading font: Calibri
- Body font: Calibri
- Heading sizes: H1 26pt bold, H2 20pt semibold, H3 15pt semibold, H4 12pt bold
- Body size: 11pt
- Line height: 1.45
- Table cell text: 10pt
- Data labels and annotations: 9pt
- Footnote size: 8.5pt

## Color palette
- Primary: #006D77 — use for headings, primary chart series, key insight borders, and section dividers
- Secondary: #83C5BE — use for secondary chart series, callout backgrounds, and progress indicators
- Accent: #E29578 — use for warnings, risk indicators, and attention-required highlights
- Body text: #2B2B2B
- Background: #FFFFFF
- Insight box background: #EDF6F7
- Table header background: #006D77 with white text
- Table alternating rows: #F5FAFA
- Chart palette: #006D77, #83C5BE, #E29578, #264653, #FFDDD2

## Layout rules
- Page size: 8.5in x 11in (US Letter)
- Margins: 0.8in top, 0.8in bottom, 0.9in left, 0.9in right
- Section spacing: 22pt before H1, 16pt before H2, 10pt before H3
- Paragraph spacing: 7pt after each paragraph
- Column structure: single column for narrative; two-column for comparative analysis and side-by-side data
- Page numbers: bottom center, 9pt
- Header: section title left-aligned, page classification right-aligned, thin #006D77 underline
- Footer: document title centered, 8pt #AAAAAA
- Data-heavy pages may use landscape orientation where noted

## Formatting conventions
- Insight callouts: teal left border (3pt #006D77), #EDF6F7 background, bold lead sentence
- SWOT and matrix frameworks: 2x2 grid layout with colored quadrant headers
- Risk indicators: traffic light system — green (#006D77), amber (#E29578), red (#C1121F)
- All quantitative claims must include source reference as inline citation or footnote
- Comparison tables: use checkmark and cross icons for feature matrices
- Process flows: numbered steps in horizontal layout with connecting arrows
- Section numbering: decimal throughout (1.0, 1.1, 2.0, 2.1)
- Gantt charts and timelines: horizontal bar format using primary and secondary palette
- Recommendations: numbered list with priority tag (High/Medium/Low) color-coded

## Tone and voice
- Analytical, evidence-based, and forward-looking
- Write as a strategy consultant presenting to a C-suite audience
- Every assertion backed by data, market evidence, or structured reasoning
- Use frameworks explicitly: "Applying Porter's Five Forces," "Using a PESTEL lens"
- Prefer "the data indicates" and "market analysis reveals" over opinion-based language
- Active voice dominant; passive acceptable for methodology descriptions
- Concise paragraphs — maximum 5 sentences per paragraph
- Use "we identify," "this analysis demonstrates," "the strategic imperative is"
- Quantify impact wherever possible: time savings, cost reduction percentages, market share shifts

## What this style MUST produce
- Strategy-grade documents with visible analytical frameworks and data backing
- At least one insight callout box per major section
- Quantitative evidence for every major recommendation
- Risk or opportunity assessment using traffic light or matrix format
- Clear numbered recommendations with priority indicators
- Comparison matrices where alternatives are evaluated
- Professional chart and data visualization styling
- Page count: per deliverable type specification
- All sections in the order specified by the deliverable module

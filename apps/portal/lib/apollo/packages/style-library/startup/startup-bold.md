# Startup Bold — Style Specification
**Version:** 1
**Industry:** Startup

## Typography
- Heading font: Poppins
- Body font: Poppins Light
- Heading sizes: H1 40pt extra-bold, H2 28pt bold, H3 20pt semibold, H4 16pt medium
- Body size: 14pt for slides, 11pt for narrative documents
- Line height: 1.6
- Stat callout size: 64pt Poppins extra-bold
- Tagline/hook size: 32pt Poppins semibold
- Caption: 10pt Poppins light

## Color palette
- Primary: #FF6B6B — use for headings, hero sections, key stats, and primary CTA buttons
- Secondary: #4ECDC4 — use for supporting data, secondary charts, success indicators, and contrast elements
- Dark: #2C3E50 — use for body text, dark slide backgrounds, and footer bars
- Light accent: #FFE66D — use for highlight markers, callout badges, and attention pops
- Body text: #2C3E50
- Background: #FFFFFF for standard slides; #2C3E50 for hero and section divider slides
- Card background: #FFF5F5
- Chart palette: #FF6B6B, #4ECDC4, #FFE66D, #45B7D1, #96CEB4

## Layout rules
- Page size: 16:9 widescreen (13.33in x 7.5in) for presentations; 8.5in x 11in for documents
- Margins: 0.5in all sides for slides (edge-to-edge feel); 0.8in for documents
- Section spacing: one concept per slide; 28pt before H1 for documents
- Content per slide: maximum 4 lines of text or one hero visual
- Column structure: bold asymmetric layouts — 60/40 splits, full-bleed images with text overlays
- Slide numbers: bottom right, 12pt bold, #FF6B6B
- No traditional headers — use full-width color bars as section dividers
- Dark slides (#2C3E50) alternate with white slides for visual rhythm

## Formatting conventions
- Hero stats: 64pt number centered with one-word descriptor below in 16pt
- Full-bleed color sections: coral (#FF6B6B) background with white text for impact statements
- Problem slide: dark background with single provocative question in 32pt white text
- Solution slide: white background with product screenshot/mockup and three benefit bullets
- Market opportunity: oversized number (64pt) with context line — "$47B and growing"
- Traction: counter-style metrics in a horizontal row (Users / Revenue / Growth Rate)
- Team: bold headshot placeholders with first-name-only labels and one-line bios
- Roadmap: three-phase horizontal timeline with milestone icons
- Testimonial/social proof: large quote marks, customer quote in 18pt italic, attribution below
- CTA/Ask: full coral background slide with white text, specific dollar amount, one line of context
- Emoji and icons used liberally for energy and quick scanning

## Tone and voice
- High-energy, direct, and unapologetically ambitious
- Write as a founder who believes deeply and has the numbers to back it up
- One-line hooks open every section: "Insurance is broken." "We fixed hiring." "Growth is not optional."
- Fragment sentences for impact. Full sentences for explanation.
- Use active, visceral verbs: "crush," "unlock," "accelerate," "dominate," "ship"
- Numbers are always impressive because you frame them right: "47 customers in 90 days"
- Do not hedge — if the data is real, state it boldly
- Second person to the audience: "You know this problem." "Your portfolio needs this."
- Humor and personality acceptable: be human, not corporate
- The ask is bold and clear: "We need $5M to own this market."

## What this style MUST produce
- High-impact pitch materials that command attention from slide one
- Alternating dark and light slides for visual energy
- At least three full-bleed color slides per deck
- Hero stat callouts at 64pt for every key number
- Provocative one-liner problem statement
- Counter-style traction metrics in horizontal layout
- Bold color palette visible on every slide
- Clear, confident ask with use-of-funds visualization
- Maximum 5 words per bullet point on any slide
- Slide count: per deliverable type specification (typically 12-20 slides)
- All sections in the order specified by the deliverable module

# Government Federal Standard — Style Specification
**Version:** 1
**Industry:** Government

## Typography
- Heading font: Arial
- Body font: Arial
- Heading sizes: H1 16pt bold, H2 14pt bold, H3 12pt bold, H4 12pt bold italic
- Body size: 12pt
- Line height: 1.5 (single-spaced body with 6pt paragraph spacing, or double-spaced if RFP requires)
- Table text: 10pt
- Header/footer text: 10pt
- Footnote size: 10pt

## Color palette
- Primary: #000000 — use for all headings, body text, and rules
- Secondary: #000000 — no color variation in strict federal standard
- Body text: #000000
- Background: #FFFFFF
- Table header background: #D9D9D9 with black text
- Table borders: #000000, 1pt solid
- Table alternating rows: #F2F2F2
- No accent colors — federal submissions must be reproducible in grayscale

## Layout rules
- Page size: 8.5in x 11in (US Letter)
- Margins: 1in all sides (unless solicitation specifies otherwise)
- Section spacing: 18pt before H1, 12pt before H2, 6pt before H3
- Paragraph spacing: 6pt after each paragraph
- Column structure: single column only — no multi-column layouts
- Page numbers: bottom center, "Page X of Y" format, 10pt
- Header: document title or solicitation number centered, 10pt
- Footer: company name left, "Page X of Y" center, date right, 10pt
- No decorative elements of any kind: no colored borders, no accent bars, no icons

## Formatting conventions
- Section numbering per FAR/solicitation structure: typically L.1, L.2 or Volume I, Section 1
- Compliance matrix cross-references: every requirement tagged with solicitation paragraph number
- Tables: full black borders, no rounded corners, header row bold with gray background
- All acronyms defined on first use: "Federal Acquisition Regulation (FAR)"
- Acronym glossary required for documents exceeding 10 pages
- Page limitations strictly observed — if solicitation says 15 pages, produce exactly 15 or fewer
- Font size restrictions strictly observed — never go below minimum specified in solicitation
- Headers repeat on every page with solicitation number
- Figures and tables numbered sequentially (Figure 1, Table 1) with descriptive captions
- Cross-references by section number: "See Section L.3.2" not "see above"

## Tone and voice
- Strictly formal, objective, and compliance-focused
- Write to the evaluation criteria — every paragraph should map to a scoring factor
- No marketing language, no superlatives, no subjective claims
- Use "the contractor shall," "the offeror proposes," "this approach ensures"
- Passive voice acceptable and often preferred: "The requirement will be met by"
- Never use first person — use company name or "the offeror"
- Quantify all experience claims: "15 years," "47 contracts," "$12M annual volume"
- Reference specific past performance by contract number where applicable
- Zero tolerance for unsupported claims — every assertion needs evidence

## What this style MUST produce
- Documents that pass federal compliance review without formatting objections
- Strict adherence to page limits, font sizes, and margin requirements
- Section numbering that mirrors solicitation structure exactly
- Compliance cross-reference matrix where applicable
- Acronym glossary for documents over 10 pages
- All tables with full borders and sequential numbering
- No color elements — entire document must be grayscale-reproducible
- Page count: per deliverable type specification and solicitation limits
- All sections in the order specified by the deliverable module or solicitation

# Legal Minimal — Style Specification
**Version:** 1
**Industry:** Legal

## Typography
- Heading font: Helvetica Neue
- Body font: Helvetica
- Heading sizes: H1 24pt medium weight, H2 18pt medium weight, H3 14pt medium weight, H4 12pt bold
- Body size: 11pt
- Line height: 1.5
- Footnote size: 9pt
- Monospace for statute references: Courier New 10pt

## Color palette
- Primary: #333333 — use for headings and all emphasis elements
- Secondary: #666666 — use for subheadings, captions, and secondary text
- Accent: #333333 — thin rules and dividers only (no color accents)
- Body text: #333333
- Background: #FFFFFF
- Table header background: #F0F0F0 with #333333 text
- Table borders: #E0E0E0
- Highlight background: #FAFAFA

## Layout rules
- Page size: 8.5in x 11in (US Letter)
- Margins: 1in all sides
- Section spacing: 28pt before H1, 20pt before H2, 14pt before H3
- Paragraph spacing: 10pt after each paragraph
- Column structure: single column exclusively — no sidebars, no multi-column
- Page numbers: bottom center, 9pt, simple numeral only
- Header: document title only, left-aligned, 8pt, #999999, starting page 2
- Footer: page number only — no confidentiality boilerplate unless client requires it
- Generous whitespace: minimum 16pt between end of section and next heading

## Formatting conventions
- Section numbering: simple numeric (1, 2, 3) with dot notation for subsections (1.1, 1.2)
- No decorative elements: no colored boxes, no sidebar accents, no pull quotes
- Horizontal rules: 0.5pt #E0E0E0 between major sections only
- Defined terms in bold, no separate definitions appendix — define inline on first use
- Legal citations in standard Bluebook, minimal formatting
- Lists: bullet points using em dash (—) character, not circles or squares
- Tables: minimal borders, header row with light gray background, no vertical rules
- Signature area: simple "___" underline with name and title below, left-aligned
- Block quotes: 0.75in indent both sides, no italic, no border

## Tone and voice
- Direct, concise, and efficient
- Strip all unnecessary verbal padding: no "it should be noted that" or "it is important to recognize"
- Active voice mandatory — passive voice only when the actor is genuinely unknown
- Short sentences preferred; maximum 35 words per sentence as a guideline
- No contractions in any document type
- State conclusions first, then provide supporting analysis
- Use "the firm advises" or direct imperatives for recommendations
- Eliminate redundant legal doublets: use "modify" not "alter and modify"

## What this style MUST produce
- Ultra-clean legal documents with maximum information density and minimum decoration
- No visual clutter: no colored boxes, no accent borders, no decorative elements
- Whitespace as the primary organizational tool between sections
- Conclusions and recommendations stated within the first page
- Em dash bullet lists for all non-sequential enumerations
- Minimal table styling with no vertical rules
- Page count: typically at the lower end of deliverable type range due to concise writing
- All sections in the order specified by the deliverable module

# Finance Audit — Style Specification
**Version:** 1
**Industry:** Finance

## Typography
- Heading font: Arial
- Body font: Arial
- Heading sizes: H1 18pt bold, H2 15pt bold, H3 12pt bold, H4 11pt bold
- Body size: 10.5pt
- Line height: 1.4
- Table text: 9.5pt
- Finding reference codes: 10pt Consolas monospace
- Footnote and disclaimer size: 8.5pt

## Color palette
- Primary: #4A4E69 — use for headings, section dividers, and primary table headers
- Secondary: #6B7094 — use for subheadings and secondary labels
- Critical finding: #9B2226 — use for material weakness indicators and critical findings
- Significant finding: #BC6C25 — use for significant deficiency indicators
- Observation: #4A4E69 — use for standard observations and improvement recommendations
- Resolved: #606C38 — use for remediated items and closed findings
- Body text: #22223B
- Background: #FFFFFF
- Finding box background: #F8F8FA
- Table header background: #4A4E69 with white text
- Table borders: #D0D0D8

## Layout rules
- Page size: 8.5in x 11in (US Letter)
- Margins: 1in all sides
- Section spacing: 18pt before H1, 14pt before H2, 8pt before H3
- Paragraph spacing: 6pt after each paragraph
- Column structure: single column for narrative; tabular layouts for findings and control matrices
- Page numbers: bottom center, "Page X of Y", 9pt
- Header: report title left, "CONFIDENTIAL" right, 9pt, #4A4E69 rule below (1pt)
- Footer: firm name left, engagement number center, date right, 8.5pt #999999
- Title page: engagement title, entity name, period under audit, report date, issuing firm

## Formatting conventions
- Finding IDs: sequential code format (e.g., FIN-2026-001, FIN-2026-002) in monospace
- Each finding structured as: ID, Title, Severity, Criteria, Condition, Cause, Effect, Recommendation, Management Response
- Severity badges: colored inline labels — MATERIAL WEAKNESS (red), SIGNIFICANT DEFICIENCY (amber), OBSERVATION (gray)
- Control matrix: table mapping control objectives to test results (Effective / Ineffective / Not Tested)
- Testing summary statistics: total controls tested, pass rate, findings by severity
- Trend comparison: current period findings vs. prior period in side-by-side table
- References to accounting standards: ASC, GAAS, PCAOB citations formatted precisely
- Workpaper cross-references: (WP-[number]) in parentheses after evidence statements
- Management response section: indented block with "Management Response:" header in italic
- Appendices: detailed testing procedures, sample selection methodology, acronym glossary

## Tone and voice
- Objective, precise, and methodical
- Write as an independent auditor — never express opinion beyond what evidence supports
- Use audit-standard language: "We noted," "Testing revealed," "Based on our examination"
- Findings stated as facts, not judgments: "The control did not operate effectively during Q2" not "Management failed"
- Recommendations are specific and actionable: "Implement automated three-way match by Q3 2026" not "Improve controls"
- Management responses recorded without editorial comment
- Risk language calibrated precisely: "material weakness" has a defined threshold, do not use casually
- No passive-aggressive phrasing — audit findings are professional, not adversarial
- Disclaimer language for scope limitations stated explicitly: "Our audit did not extend to"

## What this style MUST produce
- Audit-standard reports suitable for filing with regulators or audit committees
- Title page with engagement details
- Executive summary of findings by severity with totals
- Standardized finding format for every issue (ID, severity, criteria, condition, cause, effect, recommendation, response)
- Severity badges with consistent color coding
- Control testing matrix with pass/fail results
- Prior period comparison for repeat findings
- Workpaper cross-references for all evidence-based statements
- Acronym glossary and scope description
- Page count: per deliverable type specification
- All sections in the order specified by the deliverable module
